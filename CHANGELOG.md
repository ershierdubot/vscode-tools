# Changelog

All notable changes to the "VSCode Tools" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.3.0] - 2026-03-01

### Added

- JSON Formatter tool with format, minify, and validate features
- Activity Bar icon for quick access to VSCode Tools
- Extension icon (256x256 PNG with SVG source)

## [0.2.0] - 2026-03-01

### Added

- Project infrastructure: LICENSE, CONTRIBUTING.md, CI workflow
- Improved Parquet Viewer with search, sort, and export
- Real-time search with debouncing
- Column sorting (ascending/descending)
- CSV export functionality
- File statistics display

## [0.1.0] - 2026-03-01

### Added

- Initial release of VSCode Tools extension
- Parquet Viewer tool
- Sidebar with tools list
- Basic file loading and table display
- Mock data generation for testing
